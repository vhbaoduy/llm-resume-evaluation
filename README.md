# LLm Resume Evaluation

This project is a resume evaluation tool using language models. It provides functionality to extract information from resumes and job descriptions, evaluate job applications, and match resumes with job descriptions.

## Prerequisites

- Python >= 3.10
- Node.JS
- OpenAI API/Google AI Studio API

## Installation
Clone the repository: `https://github.com/vhbaoduy/llm-resume-evaluation`

2. Install the required dependencies: 
- Backend service: 
```
pip install poetry

poetry install
```
- Front end: 
```
cd frontend
npm install
```

3. Set up the environment variables:
   - `GOOGLE_API_KEY`: Your LLM API from Google

## Usage

1. Run the demo application: `python demo.py`

2. Provide the resume and job description as input.

3. The application will evaluate the resume and provide a match score and justification.
